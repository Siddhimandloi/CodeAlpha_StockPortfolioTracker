# CodeAlpha_StockPortfolioTracker
Python Stock Portfolio Tracker project made using Python. It helps users track stock investments, calculate total portfolio value, and manage stock data.
# Stock Portfolio Tracker

This project was created as part of the CodeAlpha Python Programming Internship.

## Project Description
A simple Python-based stock portfolio tracker that:
- Takes stock names and quantities from the user
- Uses hardcoded stock prices
- Calculates total investment value
- Optionally saves results into a text file

## Technologies Used
- Python
- Dictionary
- Loops
- File Handling

## Example Stocks
- AAPL
- TSLA
- GOOGL
- AMZN
- MSFT
